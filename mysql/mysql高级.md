# MySQL高级

## MySQL逻辑架构

1. **连接层**

   最上层是一些客户端和连接服务

2. **服务层**

   第二层架构主要完成核心服务功能, 如SQL接口, 优化器等

3. **引擎层**

   存储引擎层

4. **存储层**

   数据存储层, 主要是将数据存储在运行于裸设备的文件系统之上, 并完成与存储引擎的交互
